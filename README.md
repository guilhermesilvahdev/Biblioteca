# Biblioteca
Um sistema feito em Java que representa uma biblioteca escolar. Tendo cadastros de livros e alunos e a possibilidade de ver o livro e/ou aluno cadastrado.

## Funcionalidades
-Cadastro de livros.

-Cadastro de alunos.

-Empréstimo de livros.

-Devolução co cálculo de multa.

-Busca de livros e alunos.

-Listagem geral.

## Tecnologias
-Java.

-Programação Orientada a Objetos.

## Como executar
1.Abrir no Eclipse ou qualquer editor compativél.

2.Executar a classe Principal.

3.Utilizar o menu interativo.
